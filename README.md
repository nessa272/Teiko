# Teiko Technical
 
Reads/Writes from a given cell-count CSV file.

## Installation

### Prerequisites
- Python 3.8 or later  
- Required Python libraries:  
  - [pandas](https://pandas.pydata.org/) – `pip install pandas`  
  - [matplotlib](https://matplotlib.org/) – `pip install matplotlib`  

### Usage
Run
```sh
python teiko_technical.py
